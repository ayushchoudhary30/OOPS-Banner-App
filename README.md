# OOPS-Banner-App

OOPS Banner App - Java learning project demonstrating OOP concepts through banner display use cases.

## Getting Started

### Prerequisites

- Java JDK 11 or higher
- Maven

### Installation

1. Clone the repository:
```bash
git clone https://github.com/ayushchoudhary30/OOPS-Banner-App.git
cd OOPS-Banner-App
```

2. Install dependencies:
```bash
mvn clean install
```

3. Run the application:
```bash
mvn exec:java -Dexec.mainClass="OOPSBannerApp"
```
